# การตกแต่งตัวอักษรด้วย baffle by Javascript

> เพิ่มส่วนการทำงานของ script <br>
<script src="https://cdn.jsdelivr.net/npm/baffle@0.3.6/dist/baffle.min.js"></script><br>
<br>
> สร้างฟังก์ชั่น <br>
<script type="text/javascript"><br>
